## 2026-03-25

### oritek-world-monitor 三大问题全面修复及部署

**修复内容：**

#### 1. 地图显示问题（根本原因：D3 地图数据加载失败时无回退）
- 改进 `renderWorldMapD3()` 函数，添加多个备用数据源（本地 → jsDelivr CDN → unpkg）
- 新增 `renderFallbackMap()` 内置简化大陆地图，确保地图始终显示
- 分离 `renderHotspotMarkers()` 独立函数，热点标记在任何情况下都能渲染
- 替换 SMIL 动画为 CSS 动画（兼容性更好）

#### 2. 各版块内容不更新（根本原因：dataService 返回静态数据）
- 重写 `dataService.ts`，实现动态数据生成
- 新增 `forceRefreshAll()` 函数，强制清除缓存并重新生成所有数据
- 数据带有随机波动，模拟真实市场变化
- 新闻数据每次调用组合不同，时间戳动态更新

#### 3. 手动刷新不工作（根本原因：performFullRefresh 未更新全局变量）
- 重写 `performFullRefresh()`，调用 `forceRefreshAll()` 强制清除缓存
- 刷新时更新 `newsData`、`marketPerformance`、`globalHotspots`、`industryIndices` 等全局变量
- 地图和图表在刷新后重新渲染

#### 4. 新增后端数据抓取工具
- `scripts/data-scraper.js` - Node.js 数据抓取脚本
- `scripts/auto-refresh.bat` - Windows 自动刷新批处理  
- `scripts/setup-automation.ps1` - 计划任务设置（每10分钟）

**部署：** 已成功发布到 https://szh987210.github.io/oritek-world-monitor/
